# JAVA-PROJECT

Project Description:

This is a Java-based GUI application built using Swing and JDBC for managing events, students, and registrations.
The project provides a complete interface for event organizers to:

Add, update, and delete events
Manage student details
Record and track registrations
View all data in interactive panels

The system is designed with modular panels (EventsPanel, StudentsPanel, RegistrationsPanel) managed from the main application frame (MainFrame.java).
Database connectivity is handled through the DB.java class using the MySQL Connector/J library.

Tech Stack:

Language: Java
GUI Framework: Swing
Database: MySQL
Libraries: MySQL Connector/J
